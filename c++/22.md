### 2.动态规划

- 当我们清楚所有 i<n 时括号的可能生成排列后，对与 i=n 的情况，我们考虑整个括号排列中最左边的括号：它一定是一个左括号，那么它可以和它对应的右括号组成一组完整的括号 "( )"，我们认为这一组是相比 n-1 增加进来的括号
- G(n) = { "(" + 【i=p时所有括号的排列组合】 + ")" + 【i=q时所有括号的排列组合】}
- 其中 `p + q = n-1`，且 `p q` 均为非负整数。
